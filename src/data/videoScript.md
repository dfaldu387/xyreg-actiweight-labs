# XyReg Getting Started Video Script
*Total Duration: 8-10 minutes*

## Voice & Tone Guidelines
- **Voice**: Professional, warm, confident
- **Pace**: Moderate (140-160 words per minute)
- **Tone**: Friendly but authoritative, like a knowledgeable colleague
- **Style**: Clear enunciation, slight pauses at key points
- **Background**: Subtle ambient music, volume at 10-15%

## SECTION 3: PHASE DEPENDENCIES (90 seconds)
**[Screen: Phase management, dependency setup]**

**NARRATOR:**
"XyReg's powerful dependency engine ensures proper project sequencing. Let's explore the four dependency types."

**[Screen: Creating Finish-to-Start dependency]**

"Finish-to-Start: The most common dependency. Design Controls must finish before Risk Management begins. This ensures proper documentation flow."

**[Screen: Creating Finish-to-Finish dependency]**

"Finish-to-Finish: Clinical trials and regulatory submission preparation finish together. Both need to conclude simultaneously for submission readiness."

**[Screen: Creating Start-to-Start dependency]**

"Start-to-Start: Software development and verification planning start together. Parallel activities that must begin simultaneously."

**[Screen: Creating Start-to-Finish dependency]**

"Start-to-Finish: Less common but critical. Manufacturing scale-up starts when prototype testing finishes. This ensures resource optimization."

**[Screen: Timeline view showing all dependencies visually]**

"The timeline view shows all dependencies visually, making it easy to understand project flow and identify critical paths."

## PRODUCTION NOTES

### Visual Guidelines:
- **Screen Resolution**: 1920x1080 minimum
- **Mouse Movement**: Smooth, purposeful cursor movements
- **Click Timing**: Brief pause before each click for clarity
- **Transitions**: 0.5-second fade between sections
- **Callouts**: Subtle highlighting for key features (soft yellow glow)

### Audio Specifications:
- **Format**: 48kHz, 24-bit WAV
- **Narration**: Centered, clear foreground
- **Music**: Subtle, professional background at -20dB
- **Silence**: 0.5-second pause between major sections

### Accessibility:
- **Captions**: Full transcript with timing
- **Color Contrast**: Ensure WCAG 2.1 AA compliance
- **Text Size**: Minimum 14px for all on-screen text
- **Alternative Text**: Describe visual elements for screen readers

### Distribution Formats:
- **Web**: MP4, H.264, 1080p
- **Mobile**: MP4, H.264, 720p
- **Training**: Interactive chapters with navigation
- **Embedded**: Optimized for help system integration

### Success Metrics:
- User engagement (watch time, completion rate)
- Feature adoption post-viewing
- Support ticket reduction for covered topics
- User satisfaction scores
